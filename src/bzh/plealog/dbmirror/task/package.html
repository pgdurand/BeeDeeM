This package contains the code of the Task framework of the DBMS installation processor.

This processor is represented by class PTaskEngine which can be viewed as a FIFO stack of PTask objects. When started, PTaskEngine will execute automatically each PTask using a timer.

There are two major types of tasks:

1. PTask: a standard task

2. PParserTask: a task aims at reading a databank file (sequence or biological classification)

Stack of PTasks is prepared by the LoaderMonitor system.
