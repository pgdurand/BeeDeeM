This package contains the code of the DBMS graphical user interface.

DBMSUIStarter aims at starting DBMS UI within a JFrame. It can be used to start DBMS as a standalone application of from another software.

To setup the DBMS UI to be used as a Component to add within some existing Component, use DBMSUserInterface class. This class enables to gently setup the DBMS UI which is represented by class DBMSPanel.

InstallingDescriptorList is the class used to display the list of databank descriptors (left part of DBMS UI).

InstalledDescriptorList is the class used to display the list of installed databank (right part of DBMS UI).

LocalDBPanel is the class used to setup the Personal Databank tab. This class is a wrapper around LocalIndexInstallPanel that contains all the UI code representing the Personal Databank Installer.

RunningMirrorPanel and DBMSLogViewer are used to setup the panel located just below the list of databank descriptors.

All other classes are used to setup actions, dialogue boxes and represent business objects for the UI components.
